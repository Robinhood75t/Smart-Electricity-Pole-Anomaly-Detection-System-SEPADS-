### 📡 Smart Electricity Pole Anomaly Detection System (SEPADS)
---
🚀 Overview

The Smart Electricity Pole Anomaly Detection System (SEPADS) is an IoT-based monitoring solution designed to detect anomalies in electricity poles in real time.

This system integrates a prototype hardware device with a web-based dashboard, enabling efficient monitoring, early fault detection, and improved maintenance of electrical infrastructure.
---
### 🧠 Key Features

- ⚡ Real-time anomaly detection from pole-mounted prototype device
- 📊 Live monitoring dashboard
- 🔔 Alerts for abnormal conditions
- 🌐 Web-based interface for accessibility
- 🔌 Backend integration with hardware device
---

### 🏗️ Project Architecture

  Prototype Device  →  Backend (Node.js/Express)  →  Frontend (React)
  Prototype Device: Sends real-time data (e.g., voltage, anomalies)
  Backend: Processes data, handles API requests, and communicates with the frontend
  Frontend: Displays data visually and provides user interaction
