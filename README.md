# ⚡ Smart Electric Pole Anomaly Detection System

## 🚀 Overview

The **Smart Electric Pole Anomaly Detection System** is an intelligent solution designed to monitor electric poles in real-time and detect anomalies such as voltage fluctuations, abnormal current flow, and overheating.

This project was developed during **HackHeist 2.0 at MIET College**, showcasing a combination of IoT, web technologies, and real-time monitoring.

---

## 🎯 Problem Statement

Electric poles are critical infrastructure, but faults like:

* Voltage instability
* Excess current
* Overheating

often go unnoticed until failure occurs.

This system aims to **detect such anomalies early** and alert authorities to prevent damage and power outages.

---

## ✨ Features

* ⚡ Real-time voltage and current monitoring
* 🔥 Temperature anomaly detection
* 📡 IoT-based data transmission using ESP8266
* 🚨 Instant alerts for abnormal conditions
* 🌐 Interactive web dashboard
* 📊 Data visualization for analysis

---

## 🧠 How It Works

1. Sensors collect real-time data:

   * Voltage Sensor
   * Current Sensor
2. Data is processed using the **ESP8266 microcontroller**
3. The microcontroller sends data to the backend server
4. Backend analyzes data for anomalies
5. If abnormal behavior is detected:

   * Alerts are triggered
   * Data is logged
6. Frontend dashboard displays live data and system status

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/smart-electric-pole.git
cd smart-electric-pole
```

### 2️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

### 3️⃣ Run Backend Server

```bash
npm start
```

### 4️⃣ Setup Frontend

```bash
cd frontend

```
